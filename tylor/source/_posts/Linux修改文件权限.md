三位二进制树表示权限值:
1. 可读
2. 可写
3. 可执行

例如：101 =5，表示可读可执行，不可写

---

每个文件分3组权限，
1.所属人权限
2.同组用户权限
3.其他组用户权限

使用以下形式修改文件权限。


```
chmod -R 权限值 文件名
```
-R表示递归
对文件夹及其所有内容开放所有权限：
```
chmod -R 777 foldname
```

PS:查看文件权限命令

```
ls -l
```
